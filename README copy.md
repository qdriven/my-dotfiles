## Installation Script

```sh
sh mac/install-basic.sh
sh languages/setup.sh
```

After run this script, following components are installed:

- nodejs
- java(openjdk)
- golang
- python
  * ansible
  * docker-composer
- visual studio code(configuration,use ```code``` in terminal)
